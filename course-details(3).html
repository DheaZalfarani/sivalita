Belajar PHP untuk PEMULA | ARRAY

Array adalah sebuah variable yang bisa menampung lebih dari satu nilai. Array mirip seperti tempat penyimpanan pensil warna, dimana setiap pensil warna dianggap sebagai nilai. Array digunakan agar mudah dalam mengelola banyak nilai, sehingga tidak perlu menyimpan setiap nilainya dalam variabel terpisah. Contoh penggunaan array jika ingin mencari pensil warna merah di antara banyak pensil warna, maka dengan menggunakan array kita hanya perlu mencari di dalam array tersebut tanpa harus memeriksa satu persatu.

Membuat dan menampilkan array

Cara membuat array pada PHP dan perbedaan antara cara lama dan cara baru

Gunakan keyword array untuk membuat variabel array. Isi variabel dengan element-element yang dipisahkan oleh koma. Gunakan tanda kutip jika element bertipe string. Gunakan kurung siku [] untuk membuat variabel array. Isi variabel dengan element-element yang dipisahkan oleh koma. Tipe data dari setiap element dapat berbeda-beda. cara menampilkan isi dari sebuah array pada PHP. Menggunakan fungsi print_r() atau var_dump() untuk menampilkan isi dari sebuah array. Fungsi-fungsi tersebut akan menampilkan seluruh isi dari sebuah array beserta tipe datanya.

Menambahkan elemen pada array

cara menampilkan dan menambahkan element pada array menggunakan PHP.

Menggunakan print_r atau var_dump untuk menampilkan isi array saat proses debugging. Untuk menampilkan isi array secara rapi, kita dapat menggunakan echo dengan memanfaatkan loop seperti foreach. Index pada array dimulai dari 0. Untuk menampilkan satu element pada array, kita dapat menggunakan echo dengan menyertakan index dari element tersebut dalam kurung siku. Index dimulai dari 0. Menambahkan element baru pada array dengan cara mengakses index terakhir dan menambahkannya dengan nilai yang diinginkan.

Membuat kotak dengan PHP

Cara membuat kotak dengan menggunakan PHP

Membuat Kotak Manual Simpan angka 3 dan tulis manual untuk membuat kotak. Tambahkan style pada div agar terlihat menarik. Buat kotak sesuai isi dari eray dan tambahkan margin serta float left. Mengambil Data dari Erray Gunakan pengulangan for untuk mengambil data dari eray. Gunakan fungsi count untuk menghitung jumlah element pada eray. Menggunakan 4H Gunakan 4H untuk melakukan looping pada eray. Berikan clear pada div dan gunakan class clear supaya ada turun ke bawahnya. Buka tag PHP dan buka 4H. Simpan setiap element yang diambil ke dalam sebuah variable baru. Nama variable bebas tetapi merepresentasikan satu element dari element-array yang ada pada eray angka.
