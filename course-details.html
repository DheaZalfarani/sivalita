Belajar PHP untuk PEMULA | FUNCTION

Function adalah potongan program atau baris-baris kode yang diberi nama dan dapat dipanggil berulang-ulang. Ada 2 jenis function di PHP: build-in function (fungsi bawaan PHP) dan user-defined function (fungsi buatan pengguna). Build-in function adalah fungsi yang sudah disediakan oleh PHP seperti fungsi date dan time. User-defined function adalah fungsi yang dibuat sendiri oleh pengguna. Fungsi build-in date dan time sering digunakan dalam aplikasi web untuk menampilkan tanggal dan waktu. Beberapa fungsi build-in date dan time yang sering digunakan antara lain: time, date, mktime, dan strtotime. Untuk menggunakan fungsi date pada PHP, perlu membuat folder baru di dalam htdocs terlebih dahulu.

Contoh penerapan Unix Time Step adalah memberikan tenggat waktu diskon atau menghitung umur seseorang dari tanggal akhirnya. Manipulasi Waktu

manipulasi waktu pada PHP dan memberikan contoh penggunaannya.

Gunakan fungsi mktime() untuk menghitung detik yang sudah berlalu dari 1 Januari 1970 sampai tanggal tertentu. Fungsi ini memiliki tujuh parameter yaitu jam, menit, detik, bulan, tanggal, tahun dan zona waktu. Contoh penggunaannya: mktime(0, 0, 0, 8, 25, 1985) akan menghasilkan jumlah detik yang sudah berlalu dari tanggal tersebut hingga saat ini. Gunakan fungsi mktime() untuk membuat sendiri detiknya dari satu Januari 1970 sampai dengan tanggal lahir kita. Gunakan fungsi date() dengan parameter "l" untuk menampilkan hari dalam bahasa Inggris atau "N" untuk menampilkan hari dalam bentuk angka (1 = Senin). Contoh penggunaannya: date("l", mktime(0, 0, 0, 8, 25, 1985)) akan menampilkan hari Minggu.
